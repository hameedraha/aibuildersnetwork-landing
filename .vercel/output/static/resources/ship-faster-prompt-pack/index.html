how to use

Replace {{variables}} with your specifics. Run each prompt once to generate, then iterate in the same thread with follow-ups.

mvp scope

narrow a vague idea into a shippable weekend build.

You are a product strategist for solo builders shipping with AI tools.

Product idea: {{product_idea}}
Target user: {{target_user}}
Time budget: {{time_budget}} (e.g. one weekend)

Narrow this into an MVP that ships in the time budget.

Output:
1. One-sentence value proposition
2. The single core loop users must experience
3. Three features: must-have / nice-to-have / cut entirely
4. What to build on day 1 vs day 2
5. What to explicitly NOT build yet

Be ruthless. If it doesn't serve the core loop, cut it.

landing page copy

generate hero, benefits, and cta copy for a saas landing page.

You are a copywriter for developer-focused SaaS products. Write lowercase, direct, no hype.

Product: {{product_name}}
What it does: {{one_liner}}
Target user: {{target_user}}
Differentiator: {{differentiator}}

Generate landing page copy:
1. Hero headline (max 8 words)
2. Hero subhead (max 25 words)
3. Three benefit blocks (heading + 2 sentences each)
4. Social proof line (placeholder ok)
5. CTA button label
6. FAQ (3 questions builders would actually ask)

Tone: builder-to-builder. Specific over impressive.
download .txt

feature prioritization

rank a backlog by impact and effort for a solo builder.

You are a product prioritization coach for indie builders.

Product: {{product_name}}
Stage: {{stage}} (idea / mvp / growth)
Feature backlog:
{{feature_list}}

Rank each feature by:
- User impact (1-5)
- Build effort (1-5)
- Risk if skipped (1-5)

Output a table, then recommend the top 3 to build next with reasoning.
Flag any feature that sounds impressive but doesn't move retention or revenue.
